# Supplementary Information: EAQECCs from Generalized Extended Codes

This repository contains the table-level supplementary data for the EAQECCs retained in the final manuscript.

Each TSV file follows the row order of the corresponding paper table.  The column `Paper_Table_Row_Number` is the `No.` value in the manuscript table, so each construction record can be matched directly with the paper.

## Table-Level Data

| Paper table | Manuscript label | Description | Rows | Data |
|---|---|---|---:|---|
| Table IV | `tab:breakthroughs2140` | New and improved binary EAQECCs with lengths up to `n <= 40` | 267 | [Open TSV](tables/SI-EAQECCs_Table_IV_binary.tsv) |
| Table V | `tab:breakthroughs0118` | New and improved ternary EAQECCs with lengths up to `n <= 25` | 14 | [Open TSV](tables/SI-EAQECCs_Table_V_ternary.tsv) |
| Table VII | `tab:literature-separated-eaqeccs` | Reproduced known EAQECCs that improve Grassl's tables | 10 | [Open TSV](tables/SI-EAQECCs_Table_VII_literature_equal.tsv) |

## TSV Format

| Column | Meaning |
|---|---|
| `Paper_Table_Row_Number` | Row number in the corresponding manuscript table. |
| `Quantum_Code_Parameters` | EAQECC parameter `[[n,k,d;c]]_q` listed in the paper. |
| `Generator_Matrix_G` | Generator matrix `G` of the initial linear code `C`; rows are separated by `|`, and entries in each row are separated by spaces. |
| `Vector_u` | Extension vector `u` in the generalized extended code `C(u,a)`. |
| `Scalar_a` | Extension scalar `a` in `C(u,a)`. |

The generalized extended code is generated by the block matrix `[G 0; u a]`.

For binary EAQECCs, the listed construction data are over `GF(4)`.  For ternary EAQECCs, they are over `GF(9)`.  The symbol `w` denotes the primitive element used in the computational verification scripts.

## Scope

This repository includes only the EAQECCs appearing in the retained manuscript tables.  Additional EAQECCs from the broader search are intentionally excluded from this table-level supplementary dataset.
